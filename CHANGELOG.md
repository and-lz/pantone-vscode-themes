# Changelog

## 0.1.0 - 2026-03-01

### Added

- Initial release with 200 themes (100 Pantone colors x dark/light)
- Pantone Colors of the Year collection (2000-2025)
- Classic, neon, nature-inspired, and elegant color palettes
- HSL color theory palette generation (analogous, complementary, triadic)
- WCAG AA contrast validation with automatic adjustment
- Bold/italic syntax styling for semantic code distinction
- Full workbench theming (editor, sidebar, terminal, git decorations)
- Zero runtime dependencies
